## üöÄ How to Use

1. **Clone the repository:**
   ```bash
   git clone https://github.com/RK0297/Machine-Learning-Coursework.git
   cd Machine-Learning-Coursework
   ```

2. **Install dependencies:**
   ```bash
   pip install -r requirements.txt
   ```
   Or manually install:
   ```bash
   pip install jupyter numpy pandas scikit-learn matplotlib seaborn tensorflow keras
   ```

3. **Run Jupyter notebooks:**
   ```bash
   jupyter notebook
   ```
   Navigate to the desired folder and open any `.ipynb` file

4. **Start with basics:**
   - Begin with `Hands-on/basics.ipynb` for ML fundamentals
   - Then explore algorithm-specific notebooks in Supervised/Unsupervised folders
