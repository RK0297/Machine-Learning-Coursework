# Machine Learning Coursework

This repository contains lecture materials and lab assignments for machine learning coursework.

## Contents

### Lectures
- Course lecture PDFs and learning materials

### Labs
- ML_1.ipynb
- ML_2.ipynb
- ML_3.1.ipynb
- ML_6.ipynb
- ML.ipynb

## Overview

This coursework covers fundamental and advanced machine learning concepts, including:
- Supervised and unsupervised learning
- Model training and evaluation
- Data preprocessing and analysis
- Practical implementation through Jupyter notebooks

## Getting Started

1. Clone the repository
2. Install required dependencies
3. Open the Jupyter notebooks in the `labs/` directory to explore practical implementations

## Requirements

- Python 3.x
- Jupyter Notebook
- Common ML libraries (scikit-learn, pandas, numpy, matplotlib, etc.)
